Q.1 What is JavaScript. How to use it?

A= JavaScript is a versatile programming language primarily used for creating interactive and dynamic content on web pages. It can be used to enhance the user interface and experience by providing features such as form validation, animations, and event handling. JavaScript can be used both on the client side (in the browser) and on the server side (using environments like Node.js). It is typically embedded directly into HTML using the <script> tag.
